# Astro Documentation

This skill provides comprehensive documentation for Astro.

## How to use

The documentation is organized into categories. Start by reading the [Main Index](references/INDEX.md) or one of the category indices below to find the specific topic you are interested in.

### Categories

- [Getting Started](references/GETTING.md) - Installation and setup.
- [Tutorial](references/TUTORIAL.md) - Step-by-step tutorial (Build a Blog).
- [Basics](references/BASICS.md) - Project structure, components, pages, layouts.
- [Concepts](references/CONCEPTS.md) - Islands architecture, Astro features.
- [Guides](references/GUIDES.md) - Markdown, CMS, Deploy, Integrations, and more.
- [Recipes](references/RECIPES.md) - Short, focused how-to guides.
- [Reference](references/REFERENCE.md) - Configuration, Runtime API, Template Syntax, CLI.

### Workflow

1.  **Identify the topic**: Determine if you need a guide, a reference, or a concept explanation.
2.  **Select an Index**: Open the relevant index file (e.g., `references/REFERENCE.md` for API details).
3.  **Find the File**: Look for the specific topic in the index list.
4.  **Read the File**: Open the specific markdown file to get the details.
